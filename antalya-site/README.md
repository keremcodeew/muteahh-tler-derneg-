# Antalya Müteahhitler Derneği - Kurumsal Web Sitesi

React tabanlı kurumsal inşaat derneği web sitesi. Tasarım: navy blue, mavi ve altın vurgu renkleri; temiz, profesyonel görünüm.

## Teknolojiler

- React 18 (functional components, JSX)
- Vite
- Plain CSS (CSS değişkenleri, responsive)
- Harici UI kütüphanesi yok

## Kurulum

```bash
cd antalya-site
npm install
npm run dev
```

## Yapı

- **Navbar** – Logo, menü (Ana Sayfa, Hakkımızda, Duyurular, Projeler, Üyelik, İletişim), Giriş butonu
- **HeroSection** – Tam genişlik arka plan, slogan
- **BannerSlider** – 3 adet inşaat görselli banner kartı
- **Announcements** – Güncel Duyurular, 3 kart
- **Projects** – Projelerimiz, 3 proje kartı, "Tüm Projeler" butonu
- **MembershipCTA** – Üyelik çağrısı ve kısa testimonial
- **AboutSection** – Hakkımızda (sol görsel, sağ metin)
- **Footer** – Koyu arka plan, sosyal medya ikonları, telif

## Responsive

Desktop, tablet ve mobil ekranlara uyumlu tasarım.
